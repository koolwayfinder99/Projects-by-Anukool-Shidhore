# AI-Driven Object Detection & Image Processing

**Role:** Computer Vision Engineer
**Duration:** Oct 2024 - Feb 2025
**Tech Stack:** OpenCV, Python, Deep Learning, Image Processing

## Project Overview
Engineered a robust computer vision pipeline utilizing AI algorithms to detect and classify objects in real-time environments, specifically tailored for mechatronic applications.

## Key Technical Contributions
* **Pipeline Architecture:** Implemented a comprehensive image processing pipeline including preprocessing (noise reduction, normalization) to prepare visual data for model ingestion.
* **Model Deployment:** Deployed detection models that significantly improved accuracy and reliability in varying lighting conditions.
* **Real-Time Processing:** Optimized algorithms for low-latency inference, ensuring the system could be used for dynamic robot interaction tasks.
