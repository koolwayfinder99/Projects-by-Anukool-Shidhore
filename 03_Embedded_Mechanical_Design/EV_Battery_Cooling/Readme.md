# Hybrid Cooling System for EV Battery

**Role:** Mechanical Lead
**Duration:** Jun 2022 - Jun 2023
**Tech Stack:** ANSYS Thermal, MATLAB/Simulink, Composite Materials

## Project Overview
Led the design and experimental validation of a hybrid EV cooling system, utilizing composite materials to improve thermal efficiency.

## Key Technical Contributions
* **Thermal Analysis:** Utilized **ANSYS** to simulate heat transfer and successfully reduced battery temperatures by **13%**.
* **Control Algorithms:** Developed numerical control algorithms using **MATLAB/Simulink** to optimize cooling efficiency under varying driving cycles.
* **Validation:** Validated simulation models through a physical prototype, achieving a strong correlation between theoretical data and real-world performance.
