# IoT-Enabled Embedded Control System for Intelligent Robotics

**Role:** Embedded Systems Architect
**Duration:** Oct 2024 - Feb 2025
**Tech Stack:** ESP32, Arduino, KiCad, MPU6050 (IMU), MQTT/HTTP

## Project Overview
Architected the embedded intelligence system for an autonomous rover, integrating IoT telematics for real-time remote monitoring and control.

## Key Technical Contributions
* **Microcontroller Integration:** Utilized **ESP32** and Arduino microcontrollers to execute sensor fusion algorithms.
* **Sensor Logic:** Combined data from IMUs (**MPU6050**) and ultrasonic sensors to achieve precise obstacle avoidance and orientation tracking.
* **PCB Design:** Designed custom PCBs using **KiCad** to miniaturize electronics and ensure robust connections.
* **Protocols:** Implemented standard communication protocols (**UART, I2C, SPI**) for data transmission between sensors, actuators, and the cloud dashboard.
