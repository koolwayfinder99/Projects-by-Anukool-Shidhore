# Hardware Architecture Design for Autonomous Pollination Robot

**Role:** Systems Engineer
**Duration:** Oct 2025 - Present
**Tech Stack:** Feasibility Analysis, Component Selection, Safety Standards

## Project Overview
Designed the hardware architecture for a robotic system capable of mimicking insect behavior to transfer pollen in enclosed agricultural environments.

## Key Technical Contributions
* **Feasibility Analysis:** Conducted detailed analysis to select and justify specific sensors and actuators based on cost, weight, and precision requirements.
* **Safety & Compliance:** Ensured component selection complied with safety regulations for agricultural machinery.
* **System Integration:** Defined the interconnection map between microcontrollers, power supplies, and operational logic to validate the system's viability for long-term market implementation.
